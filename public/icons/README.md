# Icons

Drop PNG icons here before the first `npm run build`:

- `icon-16.png` (16×16)
- `icon-32.png` (32×32)
- `icon-48.png` (48×48)
- `icon-128.png` (128×128)

Until they exist, Firefox uses a placeholder and `web-ext lint` will
warn. Generate from a single SVG with `web-ext sign`, ImageMagick, or
your tool of choice. Suggested brand: lowercase "B" or a Norwegian
flag motif — keep it readable at 16px.
