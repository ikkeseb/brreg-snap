# Icons

PNG icons consumed by `manifest.json`:

- `icon-16.png` (16×16)
- `icon-32.png` (32×32)
- `icon-48.png` (48×48)
- `icon-128.png` (128×128)

Generated by `scripts/generate-icons.mjs` — downsamples
`docs/brand-b.png` (1024×1536 RGBA source) to each manifest size
using a box-filter area average. The source PNG lives outside
`public/` so it is not packed into the .xpi. Re-run after editing
the brand mark:

```
node scripts/generate-icons.mjs
```

Requires `pngjs` (devDep).
