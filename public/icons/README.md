# Icons

PNG icons consumed by `manifest.json`:

- `icon-16.png` (16×16)
- `icon-32.png` (32×32)
- `icon-48.png` (48×48)
- `icon-128.png` (128×128)

Generated by `scripts/generate-icons.mjs` — Norwegian-cross motif
rendered straight to PNG with Node built-ins, no image-processing
devDep. Re-run after editing the design:

```
node scripts/generate-icons.mjs
```
